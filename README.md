# GLWP-PDMD-code

This repository provides the XGBoost training code used to reconstruct global predawn and midday leaf water potential.

## Code

- `train_lwp.py`: trains XGBoost models for predawn leaf water potential (Ψ_PD) and midday leaf water potential (Ψ_MD).
